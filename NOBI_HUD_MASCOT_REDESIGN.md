# Nobi HUD 形象重设计

更新日期：2026-07-05

> 当前推荐方向：`Nobi Plush Scout / 诺比绒感侦察员`。这一版参考用户提供的毛绒潮玩气质，但保持原创，不复刻参考图的具体脸型、嘴型、耳朵比例和标志性组合。

## 设计结论

上一版“玻璃信标”太抽象，角色感不够，也缺少用户想要的毛绒实体感。新的 Nobi 应该更像：

```text
一个戴着 HUD 眼镜、穿着蓝白条纹衬衫的小毛绒伙伴。
它像桌面上的提醒玩偶，但气质仍然属于 Evoly 的玻璃 HUD 世界。
```

这个方向保留“潮玩、可爱、戴眼镜、轻松穿搭”的吸引力，同时把细节改成 Evoly 自己的语言：轨迹眼神、青蓝胸针、雷达耳、克制微笑。

## 视觉关键词

- 毛绒感：奶油色绒毛外圈，边缘有轻微蓬松颗粒。
- 聪明感：大号黑框 HUD 眼镜，镜片有浅蓝玻璃反光。
- 陪伴感：大眼睛、小鼻子、温和微笑，不做夸张牙齿。
- 桌面感：蓝白条纹衬衫、深色短裤、小白鞋，像可摆在桌面上的实体玩偶。
- Evoly 感：眼睛里有轨迹反光，衣服上有青蓝小轨道胸针。

## 形象结构

- 头部：圆润毛绒头套，顶部两只短兔耳/雷达耳。
- 脸部：浅奶油色面部，黑框大眼镜，深蓝眼睛。
- 身体：小比例身体，蓝白细条纹衬衫，深蓝短裤。
- 标识：胸口不是 logo，而是小型 HUD 轨道胸针，避免硬贴品牌图标。
- 表情：默认微笑、提醒专注、完成开心三种基础状态。

## 色彩映射

贴合当前 HUD 主题：

- 毛绒主色：`#F3E8D7` / `#FFF6EA`。
- HUD 主色：`#6EA8FF`。
- 雷达辅色：`#28D8C0`。
- 夜间暗部：`#0B111B` / `#142033`。
- 服装蓝：`#A9D4F7` / `#243A56`。
- 玻璃高光：白色 45%-75% 透明度。

## 使用边界

适合：

- 今日页或迷你面板里的 Nobi 头像。
- 空状态插画。
- 完成任务后的轻量反馈。
- 设置页“关于 Nobi”。
- 后续做启动页或品牌介绍里的实体玩偶渲染。

不适合：

- 替代 Evoly App 图标。
- 高频出现在每条任务旁边。
- 说教式、鸡血式提示文案。
- 过度潮玩化，变成和生产力软件割裂的收藏玩偶。

## 交互建议

- 默认：眼镜高光轻微扫过，周期 3s 左右。
- 提醒：耳朵小雷达点亮一次，眼睛里的轨迹点轻闪。
- 完成：眼睛弯起，胸针亮点向上跳一下。
- 减少动态模式：只保留静态状态切换。

## 与旧方向的关系

- 保留早期 Nobi 的“伙伴”定位。
- 放弃上一版水滴形玻璃精灵轮廓。
- 吸收参考图的毛绒、眼镜、穿搭、潮玩实体感。
- 用雷达耳、轨迹眼神和 HUD 胸针建立 Evoly 自己的识别点。
