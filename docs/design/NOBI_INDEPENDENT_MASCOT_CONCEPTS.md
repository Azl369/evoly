# Nobi 独立卡通形象 10 份参考稿

> 本轮重新定义：Nobi 不需要和 Evoly App 图标产生视觉关联，也不需要胸口带上升轨迹。Nobi 是一个独立 IP 角色，核心目标是 **足够可爱、足够有记忆点、适合后续做表情包和产品陪伴角色**。

> 2026-07-05 更新：当前 Evoly 已转向 HUD 玻璃视觉语言，新的推荐方向请看 [`NOBI_HUD_MASCOT_REDESIGN.md`](NOBI_HUD_MASCOT_REDESIGN.md)。本文件保留为独立 IP 早期探索参考。

---

## 设计目标

Nobi 应该像一个会陪用户完成目标的小伙伴，但它不必长得像任务、计划、箭头或图标。

### 核心关键词

- 可爱
- 软萌
- 有一点怪
- 亲近
- 没有压迫感
- 可做表情包
- 小尺寸也能认出轮廓

### 不做的方向

- 不做成效率工具图标。
- 不做成 AI 助手机器人。
- 不做成严肃教练。
- 不强行绑定 Evoly 上升轨迹。
- 不照搬任何现有潮玩角色。

## 10 份参考稿

### 01 Bunny Monster / 兔耳小怪

文件：`design/mascot/nobi_independent/nobi-cute-01-bunny-monster.svg`

特点：

- 粉色系。
- 长耳朵。
- 圆身体。
- 表情软萌。

优点：

- 可爱度最高。
- 轮廓记忆点强。
- 很适合贴纸、头像、完成任务庆祝动画。

风险：

- 粉色方向可能偏女性化或偏幼。

推荐度：★★★★★

---

### 02 Cloud Pup / 云朵小狗

文件：`design/mascot/nobi_independent/nobi-cute-02-cloud-pup.svg`

特点：

- 蓝色云朵感。
- 像小狗又像软软的云。
- 气质温和。

优点：

- 非常亲近。
- 和“陪伴”感很匹配。
- 蓝色系和 Evoly 产品色不会冲突，但不强绑定图标。

风险：

- 怪可爱程度不如 01。

推荐度：★★★★☆

---

### 03 Pudding Sprite / 布丁精灵

文件：`design/mascot/nobi_independent/nobi-cute-03-pudding-sprite.svg`

特点：

- 黄色布丁感。
- 圆润、软、暖。
- 顶部有小角/小耳朵。

优点：

- 温暖积极。
- 很适合做“完成了，奖励一下”的情绪反馈。
- 和任务压力形成反差，比较治愈。

风险：

- 和成长/效率关联较弱，但这次不要求关联。

推荐度：★★★★☆

## 推荐排序

### 第一推荐：01 Bunny Monster

理由：

- 最可爱。
- 最有潮玩轮廓。
- 最适合做独立 IP。
- 后续表情包空间很大。

适合设定：

```text
Nobi 是一只软软的小怪兽，看起来有点调皮，但其实特别会陪人。
```

### 第二推荐：02 Cloud Pup

理由：

- 陪伴感最强。
- 蓝色系不会脱离 Evoly，但也不会绑定 App 图标。
- 更适合作为长期日常伙伴。

适合设定：

```text
Nobi 像一朵会走路的小云，轻轻地陪你把今天过完。
```

### 第三推荐：09 Candy Imp

理由：

- 更有个性。
- 更像潮玩角色。
- 可以做一些轻吐槽、轻鼓励的 UI 文案。

适合设定：

```text
Nobi 有点调皮，会在你拖延时眨眨眼：要不要先做 5 分钟？
```

## 下一轮建议

从 01 / 02 / 09 中选一个主方向，然后做第二轮精修：

1. 统一最终角色名：`Nobi / 诺比`。
2. 做三视图：
   - 正面
   - 侧面
   - 背面
3. 做 5 个基础表情：
   - 默认微笑
   - 提醒
   - 开心庆祝
   - 思考
   - 安慰
4. 做 App 内使用场景：
   - 今日建议卡头像
   - 空状态插画
   - 完成任务庆祝
   - 通知图标/通知大图
