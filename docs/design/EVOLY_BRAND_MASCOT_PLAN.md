# Evoly 品牌与卡通伙伴设计方案稿

> 当前阶段只确定设计方向，不改代码、不替换正式图标资源。后续确认后再进入图标精修、角色视觉稿、UI 文案替换和资源接入。

---

## 1. App 图标方向确认

### 选定方向：方案 B「Trajectory / 上升轨迹」

源稿：

`design/icons/evoly/evoly-icon-concept-b-trajectory.svg`

### 选择理由

- Evoly 的核心不是单纯记录，而是推动用户“往前走”。
- 上升轨迹比 checklist 更有成长感，也比指南针更直接。
- 轨迹天然适合表达：
  - 今日推进
  - 目标进度
  - 成长曲线
  - 从混乱到变好的路径

### 后续精修方向

当前 B 版还只是概念稿，后续应继续调整：

- 弱化商业增长感，避免像数据分析 App。
- 保留上升轨迹，但让线条更柔和、更像“成长路径”。
- 中间的勾选可以保留，但比例应更轻，不要抢走轨迹主视觉。
- 颜色继续使用 Evoly Blue + Soft Blue + Growth Green。

推荐最终图标关键词：

```text
向上、轻快、成长、完成感、温和推进
```

## 2. 「Coach」命名调整

### 问题判断

`Coach` 虽然准确，但有几个问题：

- 听起来偏工具化、训练感强。
- 中文语境里容易让人想到“教练”，有一点被监督、被要求的感觉。
- 和 Evoly 想要的“亲和陪伴、轻轻推你一下”不完全一致。

### 命名原则

新名字应该满足：

- 亲切，不压迫。
- 能承接“建议 / 陪跑 / 提醒 / 鼓励”。
- 不要太 AI 味，不要像冷冰冰的助手。
- 可以作为一个小角色的名字，而不只是功能模块名。
- 中英文都能读顺。

---

## 3. 名字候选

### 主推：Lumo

中文昵称：**路萌**

读法：

- 英文：Lumo
- 中文：路萌

含义：

- `Lu / 路`：路线、路径、下一步。
- `Mo / 萌`：小而亲切，有陪伴感。
- `Lumo` 也有 light / luminous 的联想，像一个小小的亮点，提醒你往前走。

适合 UI 文案：

- `Lumo 今天建议`
- `路萌提醒你`
- `路萌觉得今天可以先做这 3 件`
- `来自 Lumo 的小建议`

优点：

- 比 Coach 亲和很多。
- 可以自然延展成卡通角色。
- 与 Evoly 的成长、路径、轻提醒都匹配。
- 英文短，适合未来国际化。

风险：

- 需要通过角色视觉建立记忆点。
- 中文“路萌”略可爱，后续文案要控制分寸，避免过幼。

---

### 候选 2：Momo

中文昵称：**默默**

含义：

- 默默陪你完成目标。
- 不强迫、不训话，只是在旁边轻轻提醒。

优点：

- 中文亲和力强。
- 很适合陪伴型角色。

风险：

- 名字泛用度高，独特性不够。
- 和“目标推进”的关联弱一些。

### 候选 3：Nobi

含义：

- 有一点小怪兽、小伙伴的感觉。
- “No big pressure”的轻松感可以做文案延展。

优点：

- 角色感强。
- 可爱但不一定幼稚。

风险：

- 和 Evoly 的成长路径关联需要后续视觉补足。

---

### 候选 4：Pipi

中文昵称：**皮皮**

含义：

- 活泼、轻推、带点调皮。

优点：

- 很亲切，容易记住。

风险：

- 偏幼，可能降低产品的成长感和专业感。

## 4. 推荐命名结论

### 建议采用：Lumo / 路萌

理由：

- 比 Coach 更有温度。
- 仍然保留“方向、路径、照亮下一步”的含义。
- 可以既可爱又不幼稚。
- 和方案 B 的“上升轨迹”图标方向可以形成统一：

```text
Evoly = 个人成长路径工具
Lumo = 路上的小伙伴，帮你看见下一步
```

## 5. 卡通伙伴角色设定

### 角色名称

**Lumo / 路萌**

### 角色定位

Lumo 是 Evoly 里的“小成长伙伴”，不是教练，不是老师，也不是 AI 管家。

它的职责是：

- 帮你看见今天最该先做什么。
- 当任务太多时，提醒你收缩范围。
- 当你拖延时，帮你把下一步拆小。
- 当你完成时，给你一点轻轻的正反馈。

一句话设定：

```text
Lumo 是一只住在成长轨迹里的小伙伴，会在你卡住时，轻轻点亮下一步。
```

## 6. 视觉设定

### 形象方向

参考用户提到的 Labubu：不是照搬，而是借鉴“有记忆点的小怪兽/小伙伴”方向。

Lumo 应该是：

- 小小的、圆润的、有一点怪但不吓人。
- 有大眼睛或发光眼神，表达“看见下一步”。
- 头顶或身体有一个小小的上升轨迹符号。
- 耳朵/触角可以像箭头、叶芽或小闪电，呼应成长推进。
- 身体颜色以 Evoly Blue 为主，点缀 Growth Green。

### 基础造型

```text
轮廓：圆角小怪兽 / 小精灵
头部：略大，亲和
眼睛：大而亮，带一点好奇感
耳朵：两只短耳，形状像向上的小箭头
身体：短小，底部圆润
标识：胸口或额头有一条上升轨迹
表情：鼓励、机灵、但不鸡血
```

### 色彩

- 主体：`#5B6CFF`
- 阴影：`#3C46B8`
- 高光：`#8EA0FF`
- 成长点缀：`#2EE6A6`
- 眼睛/脸部：`#FFFFFF` + `#101828`

### 情绪版本

后续可做 5 个基础表情：

1. **默认**：微笑，适合今日建议卡。
2. **提醒**：举手/发光，适合任务提醒。
3. **庆祝**：星星眼/小跳跃，适合完成任务。
4. **思考**：歪头，适合分析任务过多。
5. **安慰**：抱着小轨迹，适合延期或失败复盘。

## 7. UI 中替换 Coach 的命名建议

### 当前

```text
Evoly Coach 今日建议
Coach 调整草案
Coach 建议
```

### 建议替换为

```text
Lumo 今日小建议
Lumo 调整草案
来自 Lumo 的提醒
路萌觉得可以先做这 3 件
```

### 文案语气

避免：

```text
你今天任务过载了
你应该先完成最高优先级任务
你拖延太多了
```

建议：

```text
今天的任务有点满，Lumo 建议先收住范围。
先从这 3 件开始，会更容易进入状态。
这个目标最近有点卡住，我们把下一步拆小一点。
今天已经完成啦，要不要简单记一下做对了什么？
```

## 8. Lumo 与 App 图标 B 的关系

图标 B 是产品主标识，表达 Evoly 的“成长轨迹”。

Lumo 是产品内部角色，表达“陪你走这条轨迹的小伙伴”。

二者关系：

```text
图标：轨迹
角色：住在轨迹里的小伙伴
功能：帮你决定下一步
```

在视觉上可以统一：

- Lumo 额头或胸口有图标 B 的上升轨迹。
- Lumo 的耳朵像上升箭头。
- Lumo 的脚下可以有一条小轨迹，作为插画场景元素。

  2. 小星球精灵

- 选定后再扩展表情包。

- 将 `Coach` 模块展示名改为 `Lumo`。
- 保留代码层 feature 名 `coach`，避免短期大重构。
- UI 层文案先替换：
  - `Evoly Coach 今日建议` → `Lumo 今日小建议`
  - `Coach 调整草案` → `Lumo 调整草案`

- 今日页建议卡左侧加入 Lumo 小头像。
- 空状态加入 Lumo 插画。
- 完成任务后给出轻量庆祝反馈。
- 通知文案中可使用 `Lumo 提醒你`。
