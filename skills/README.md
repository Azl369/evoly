# Project Skills

This directory contains project-local Codex skills and reference packs that
should travel with the Evoly repository.

## Skills

- `evoly-dev-environment/` - Evoly development environment, Flutter build,
  Android debugging, Windows build, and D-drive tooling rules.
- `evoly-android-edit-performance/` - Android real-device edit-mode,
  keyboard, bottom-sheet, and heavy-list performance diagnosis and fix
  playbook for Evoly.
- `openmontage/` - OpenMontage video-production skill pack, fixed as local
  references for montage, video workflows, prompts, editing, composition,
  subtitles, audio, and media review.
- `ui-ux-pro-max/` - UI/UX Pro Max design intelligence skill pack for
  interface design, design systems, colors, typography, accessibility,
  stack-specific UI guidance, slides, branding, and visual QA.

OpenMontage runtime tools are not vendored. See
`openmontage/PROVENANCE.md` for source, commit, copied paths, omitted runtime
parts, and license notes.

UI/UX Pro Max CLI distribution scaffolding is not vendored. See
`ui-ux-pro-max/PROVENANCE.md` for source, commit, copied paths, omitted parts,
and license notes.
