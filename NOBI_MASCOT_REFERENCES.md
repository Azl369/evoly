## 角色定位

Nobi 是 Evoly 里的成长伙伴，替代原本偏工具化的 `Coach`。

它不是教练，不是老师，也不是严肃 AI 助理，而是：

```text
一个在你目标路线上陪跑的小伙伴。
它不催你冲刺，只是在你卡住时轻轻推一下。
```

### Nobi 的性格关键词

- 亲近
- 机灵
- 温和
- 有一点怪可爱
- 不幼稚
- 不说教
- 不制造压力

### 与 Evoly 图标 B 的关系

App 图标 B 是“上升轨迹”，表达 Evoly 帮用户推进目标。

Nobi 的身体或胸口保留上升轨迹元素，表达：

```text
Nobi 住在成长轨迹里，帮你看到下一步。
```

## 方案 01：Rounded Monster / 圆角小兽

文件：`design/mascot/nobi/nobi-reference-01-rounded-monster.svg`

### 视觉特点

- 圆润身体。
- 两只上扬耳朵。
- 大眼睛。
- 胸口有 Evoly 上升轨迹。

### 优点

- 最接近用户提到的“类似 Labubu 的卡通形象”方向，但没有照搬。
- 怪可爱、有记忆点。
- 适合做表情包和 UI 插画。

### 风险

- 如果耳朵和怪兽感继续加强，可能会变得偏潮玩，和效率产品略有距离。

### 适合场景

- 今日建议卡头像。
- 空状态插画。
- 完成任务后的庆祝动效。

## 方案 02：Orbit Sprite / 轨迹精灵

文件：`design/mascot/nobi/nobi-reference-02-orbit-sprite.svg`

### 视觉特点

- 更像漂浮在轨迹里的小精灵。
- 深色背景，高级感更强。
- 身体带轨道/星球感。

### 优点

- 和 Evoly 的“成长轨迹”绑定最强。
- 更有未来感，适合后续智能建议功能。
- 不像普通待办 App 的吉祥物。

### 风险

- 亲和力略低于方案 01。
- 小尺寸下轮廓可能不如圆角小兽直观。

### 适合场景

- Lumo/Nobi 智能建议模块。
- 周复盘、成长洞察。
- 高级版品牌视觉。

## 方案 03：Planet Goblin / 星球小怪

文件：`design/mascot/nobi/nobi-reference-03-planet-goblin.svg`

### 视觉特点

- 圆形星球身体。
- 耳朵像小角，也像信号触角。
- 整体更软萌。

### 优点

- 很亲切，圆形轮廓适合做头像。
- 适合移动端卡片里的小插图。
- 可爱但不太有攻击性。

### 风险

- 个性略弱，需要后续通过表情和动作加强记忆点。

### 适合场景

- 通知提醒头像。
- 首页空状态。
- 新用户引导。

## 方案 04：Bean Buddy / 豆豆伙伴

文件：`design/mascot/nobi/nobi-reference-04-bean-buddy.svg`

### 视觉特点

- 豆子形身体。
- 轮廓最简洁。
- 表情温和，适合长期陪伴。

### 优点

- 最轻量，最不打扰。
- UI 中放小尺寸也比较稳。
- 适合 Evoly 如果想走“克制、简洁”的方向。

### 风险

- 记忆点不如 01 和 02。
- 潮玩感弱。

### 适合场景

- 小头像。
- 设置页、提醒页的小状态图标。
- 更成熟克制的品牌路线。

---

## 方案 05：Hoodie Guide / 兜帽旅伴

文件：`design/mascot/nobi/nobi-reference-05-hoodie-guide.svg`

### 视觉特点

- 深色兜帽外轮廓。
- 蓝紫脸部。
- 像一个安静的小向导。

### 优点

- “陪你走路”的感觉最强。
- 比普通可爱怪兽更有神秘感和角色感。
- 适合做 Evoly 的成长向导。

### 风险

- 深色兜帽可能略显严肃。
- 如果处理不好，会偏游戏 NPC，而不是效率产品伙伴。

### 适合场景

- 目标详情页提示。
- 长期目标陪跑。
- 复盘和建议类场景。

## 我的推荐排序

### 第一推荐：方案 01 圆角小兽

理由：

- 最符合“类似 Labubu 的卡通形象”这个方向。
- 亲和力强，有潮玩感，但仍能和 Evoly 的上升轨迹结合。
- 后续做表情、动作、通知图标都比较自然。

### 第二推荐：方案 02 轨迹精灵

理由：

- 最贴合 Evoly 的成长轨迹和未来智能建议。
- 更有品牌差异化。
- 适合如果你想让 Nobi 看起来更“聪明”和更未来感。

### 第三推荐：方案 05 兜帽旅伴

理由：

- 有陪伴和向导感。
- 适合长期目标和复盘场景。
- 但需要进一步柔化，否则会略严肃。

## 建议的下一轮精修

如果继续推进，建议从 01 / 02 中选一个主方向，再做第二轮：

1. 出 3 个表情：
   - 默认微笑
   - 提醒举手
   - 完成庆祝
2. 出 2 个体态：
   - 半身头像
   - 全身站姿
3. 测试小尺寸：
   - 32px
   - 48px
   - 72px
4. 与 UI 卡片融合：
   - 今日建议卡左侧头像
   - 空状态插画
   - 通知小图标

## 命名备注

如果确定使用 `Nobi`，中文昵称建议：

### 主推：诺比

优点：

- 简单、顺口。
- 不幼稚。
- 有小伙伴/小怪兽感。

### 候选：糯比

优点：

- 更软萌，和圆润形象更贴。

风险：

- 稍微偏幼。

### 候选：挪比

优点：

- 有“挪动一点点”的隐喻，适合目标推进。

风险：

- 字面不如“诺比”好看。

当前建议：

```text
英文名：Nobi
中文名：诺比
产品文案：Nobi 今日小建议 / 诺比提醒你
```
